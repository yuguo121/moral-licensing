# 盈余管理测量方法对比分析报告

## 一、方法差异对比表

### 1.1 核心方法对比总览


| 学者/文献              | 年份   | 模型类型                | 核心创新        | 主要变量               |
| ------------------ | ---- | ------------------- | ----------- | ------------------ |
| **Jones**          | 1991 | 第一代应计模型             | 开创性分解应计项目   | ΔREV, PPE, TA      |
| **Dechow et al.**  | 1995 | Modified Jones      | 控制应收账款操纵    | ΔREV-ΔREC, PPE     |
| **Dechow et al.**  | 1996 | Total Accruals      | SEC执法行动分析   | 总应计+异常应计           |
| **Kothari et al.** | 2005 | Performance-matched | 控制业绩偏差      | ROA, 配对方法          |
| **Roychowdhury**   | 2006 | REM模型               | 真实活动操纵      | CFO, PROD, DISX    |
| **Dechow et al.**  | 2012 | 预测模型                | F-score预测误报 | 多指标综合              |
| **Heese et al.**   | 2023 | 综合方法                | DiD+双重测量    | Modified Jones+REM |


---

### 1.2 详细回归方程对比

#### (1) Jones (1991) 原始模型

```
TA_t/A_{t-1} = α₀(1/A_{t-1}) + α₁(ΔREV_t/A_{t-1}) + α₂(PPE_t/A_{t-1}) + ε_t

其中:
- TA_t = 总应计 = (Δ流动资产 - Δ现金) - (Δ流动负债 - Δ短期债务) - 折旧
- ΔREV_t = 营业收入变化
- PPE_t = 固定资产原值
- A_{t-1} = 滞后一期总资产
- ε_t = 残差 = 操纵性应计(DA)
```

**关键假设**: 

- 营业收入变化反映经济环境，是客观的经营指标
- 非操纵性应计在估计期和事件期保持稳定

---

#### (2) Dechow et al. (1995) Modified Jones模型

```
TA_t/A_{t-1} = α₀(1/A_{t-1}) + α₁[(ΔREV_t - ΔREC_t)/A_{t-1}] + α₂(PPE_t/A_{t-1}) + ε_t

新增变量:
- ΔREC_t = 应收账款变化

关键改进:
- 用(ΔREV - ΔREC)替代ΔREV
- 假设所有信用销售变化都可能是盈余管理
```

**核心创新**: 

- 管理者更容易通过信用销售确认来操纵盈余
- 现金销售较难操纵，因此从收入变化中剔除应收账款变化

---

#### (3) Kothari et al. (2005) Performance-matched模型

**方法一: 回归法(加入ROA)**

```
TA_t/A_{t-1} = b₀ + b₁(1/A_{t-1}) + b₂(ΔREV_t/A_{t-1}) + b₃(PPE_t/A_{t-1}) + b₄(ROA_t) + e_t

新增变量:
- ROA_t = 资产收益率 = 净利润/平均总资产
- 加入常数项b₀控制异方差
```

**方法二: 配对法**

```
DA_performance-matched = DA_firm - DA_matched

配对标准:
- 同两位数SIC行业代码
- 同年份
- 最接近的ROA值
```

**核心创新**:

- 解决极端业绩公司的测度偏差
- 应计与业绩高度相关，需控制业绩影响

---

#### (4) Roychowdhury (2006) 真实活动盈余管理模型

**模型A: 异常经营现金流(CFO)**

```
CFO_t/A_{t-1} = α₀ + α₁(1/A_{t-1}) + β₁(S_t/A_{t-1}) + β₂(ΔS_t/A_{t-1}) + ε_t

其中:
- CFO_t = 经营活动现金流
- S_t = 销售收入
- ΔS_t = 销售收入变化

异常CFO = 实际CFO - 预测CFO
```

**模型B: 异常生产成本(PROD)**

```
PROD_t/A_{t-1} = α₀ + α₁(1/A_{t-1}) + β₁(S_t/A_{t-1}) + β₂(ΔS_t/A_{t-1}) + β₃(ΔS_{t-1}/A_{t-1}) + ε_t

其中:
- PROD_t = 销售成本 + 存货变化

异常PROD = 实际PROD - 预测PROD
```

**模型C: 异常可自由支配费用(DISX)**

```
DISX_t/A_{t-1} = α₀ + α₁(1/A_{t-1}) + β(S_{t-1}/A_{t-1}) + ε_t

其中:
- DISX_t = 广告费用 + 研发费用 + SG&A费用

异常DISX = 实际DISX - 预测DISX
```

**综合REM指标**:

```
REM = Abnormal_PROD - Abnormal_CFO - Abnormal_DISX
```

**核心创新**:

- 捕捉应计项目之外的操纵方式
- 真实活动操纵直接影响现金流
- 三种操纵手段: 销售操纵、生产操纵、费用削减

---

#### (5) Dechow et al. (2012) 预测模型方法

**F-score模型**:

```
F-score = -7.893 + 0.790×RSST_ACC + 2.518×WC_ACC + ...

其中关键指标:
- RSST_ACC = 应计项目变化
- WC_ACC = 营运资本应计
- 软资产比例
- 发行股票
- 销售、总费用及行政费用变化
- 经营租赁
- 审计师变更
```

**核心创新**:

- 使用Logistic回归预测财务重述概率
- 多维度指标综合评估
- 提高检测能力

---

#### (6) Heese et al. (2023) 综合方法

**应计基础盈余管理**:

```
使用Modified Jones模型(Dechow et al., 1995)
DA = 操纵性应计残差
```

**真实活动盈余管理**:

```
使用Roychowdhury(2006)模型
REM = 异常可自由支配费用(DISX)
```

**DiD设计**:

```
Treatment: CEO/CFO需要宣誓诚信的公司
Control: 不需要宣誓的公司

检验: 宣誓前后盈余管理变化差异
```

**核心创新**:

- 同时使用两种盈余管理测量
- 检验替代效应(Substitution effect)
- 政策评估的准实验设计

---

### 1.3 方法特征对比矩阵


| 特征        | Jones  | Modified Jones | Kothari | Roychowdhury | Dechow 2012 | Heese |
| --------- | ------ | -------------- | ------- | ------------ | ----------- | ----- |
| **操纵类型**  | 应计     | 应计             | 应计      | 真实活动         | 综合          | 应计+真实 |
| **数据要求**  | 财务报表   | 财务报表           | 财务报表    | 现金流量表        | 多源数据        | 完整报表  |
| **样本量要求** | 行业年≥10 | 行业年≥10         | 行业年≥10  | 行业年≥10       | 大规模         | 配对设计  |
| **估计方法**  | 时间序列   | 截面回归           | 截面+配对   | 截面回归         | Logistic    | DiD   |
| **现金流影响** | 无      | 无              | 无       | 有            | 有           | 有     |
| **审计可检测** | 是      | 是              | 是       | 否            | 是           | 部分    |
| **长期影响**  | 有限     | 有限             | 有限      | 显著           | 显著          | 显著    |

## 二、差异原因分析

### 2.1 研究问题的差异


| 文献                      | 核心研究问题             | 方法选择的逻辑                    |
| ----------------------- | ------------------ | -------------------------- |
| **Jones (1991)**        | 进口救济调查期间管理者是否低估盈余？ | 需要检测特定事件期间的盈余管理，因此开发事件研究方法 |
| **Dechow (1995)**       | 哪种模型最能检测盈余管理？      | 比较多种模型，发现应收账款是重要操纵渠道       |
| **Dechow (1996)**       | 盈余操纵的动机和后果是什么？     | 使用被SEC调查的公司，需要可靠的操纵测量      |
| **Kothari (2005)**      | 业绩极端公司是否导致测度偏差？    | 发现应计与业绩相关，需控制ROA           |
| **Roychowdhury (2006)** | 管理者是否通过真实活动操纵盈余？   | 应计之外还有其他操纵方式，需开发新模型        |
| **Dechow (2012)**       | 如何预测会计误报？          | 从检测到预测，需要更全面的指标体系          |
| **Heese (2023)**        | 诚信誓言能否减少盈余管理？      | 政策评估需要准实验设计，需检验替代效应        |


### 2.2 数据可得性的影响


| 时期             | 数据特点       | 方法限制                      |
| -------------- | ---------- | ------------------------- |
| **1991年前**     | 现金流量表非强制披露 | Jones模型使用资产负债表法计算应计       |
| **1995年后**     | 现金流量表普及    | Modified Jones可使用更精确的应计计算 |
| **2000年后**     | 财务数据更丰富    | 可开发多指标预测模型                |
| **SOX后(2002)** | 内部控制数据可得   | 可研究内控与盈余管理关系              |
| **现代**         | 大数据+机器学习   | 可开发更复杂的预测模型               |


### 2.3 样本特征的影响


| 研究                      | 样本特征        | 方法适应性     |
| ----------------------- | ----------- | --------- |
| **Jones (1991)**        | 23家进口救济公司   | 小样本事件研究   |
| **Dechow (1995)**       | SEC调查公司+对照组 | 需要高检测力模型  |
| **Kothari (2005)**      | 全样本，含极端业绩公司 | 需控制业绩偏差   |
| **Roychowdhury (2006)** | 全样本         | 需捕捉各种操纵方式 |
| **Heese (2023)**        | 荷兰公司，政策变化   | 需要DiD设计   |


### 2.4 研究时期的差异


| 时期          | 制度背景    | 方法演进        |
| ----------- | ------- | ----------- |
| **1990年代**  | 监管较宽松   | 基础模型开发      |
| **2000年代初** | 安然事件后   | 更严格的盈余管理检测  |
| **SOX后**    | 监管加强    | 从应计转向真实活动研究 |
| **2010年代**  | 金融危机后   | 预测模型和综合方法   |
| **2020年代**  | ESG+数字化 | 更复杂的政策评估设计  |

## 三、方法选择的目的

### 3.1 每种方法最适合的研究场景


| 方法                 | 最适合场景         | 不推荐场景      |
| ------------------ | ------------- | ---------- |
| **Jones (1991)**   | 早期研究、教学示例     | 现代研究(已被改进) |
| **Modified Jones** | 一般性盈余管理研究     | 极端业绩样本     |
| **Kothari PM**     | 包含极端业绩的样本     | 业绩分布均匀的样本  |
| **Roychowdhury**   | 研究SOX后、审计严格环境 | 仅需研究应计操纵   |
| **Dechow 2012**    | 预测财务困境/重述     | 一般盈余管理水平测量 |
| **Heese式综合**       | 政策评估、制度变化     | 简单横截面比较    |


### 3.2 检验效力的考虑


| 方法                 | 检测效力   | 第一类错误 | 第二类错误 |
| ------------------ | ------ | ----- | ----- |
| **Jones**          | 中等     | 中等    | 较高    |
| **Modified Jones** | 较高     | 较低    | 中等    |
| **Kothari PM**     | 高      | 低     | 低     |
| **Roychowdhury**   | 高(REM) | 低     | 中等    |
| **Dechow 2012**    | 很高(预测) | 低     | 低     |


### 3.3 稳健性考虑

**推荐的稳健性检验组合**:

1. **基础检验**: Modified Jones模型
2. **业绩控制**: Kothari Performance-matched模型
3. **真实活动**: Roychowdhury模型
4. **替代测量**: 使用会计重述作为替代指标

**稳健性检验的目的**:

- 确保结果不依赖于特定模型
- 检验应计与真实活动操纵的替代关系
- 验证研究结论的可靠性

## 四、总结与建议

### 4.1 方法演进脉络

```
Jones (1991) 
    ↓
Dechow (1995) Modified Jones ← 控制应收账款操纵
    ↓
Kothari (2005) Performance-matched ← 控制业绩偏差
    ↓
Roychowdhury (2006) REM ← 扩展至真实活动
    ↓
Dechow (2012) F-score ← 预测导向
    ↓
Heese (2023) 综合方法 ← 政策评估
```

### 4.2 实践建议


| 研究目的     | 推荐方法                        | 理由       |
| -------- | --------------------------- | -------- |
| 一般盈余管理研究 | Modified Jones + Kothari PM | 稳健性最高    |
| SOX后研究   | Roychowdhury REM            | 捕捉监管变化影响 |
| 预测研究     | Dechow (2012) F-score       | 预测能力最强   |
| 政策评估     | Heese式DiD+双重测量              | 因果推断可靠   |
| 极端业绩样本   | Kothari PM                  | 控制业绩偏差   |


### 4.3 关键要点

1. **没有完美模型**: 每种方法都有其假设和局限性
2. **稳健性至关重要**: 应使用多种方法进行交叉验证
3. **研究问题决定方法**: 方法选择应与研究问题匹配
4. **制度背景重要**: 考虑监管环境和数据可得性
5. **替代效应**: 应计与真实活动操纵可能存在替代关系
