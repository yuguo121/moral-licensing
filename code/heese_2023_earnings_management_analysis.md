# Heese et al. (2023) 盈余管理测度方法详细分析

## 论文信息

- **标题**: When Executives Pledge Integrity: The Effect of the Accountant's Oath on Firms' Financial Reporting
- **作者**: Jonas Heese, Gerardo Pérez-Cavazos, Caspar David Peter
- **期刊**: The Accounting Review, 2023, 98(7): 261-288
- **DOI**: 10.2308/TAR-2021-0600

---

## 一、研究设计背景

### 1.1 荷兰诚信誓言制度

**制度背景**：

- 2016年5月17日，荷兰皇家特许会计师协会（NBA）董事会要求所有持有荷兰专业会计学位（类似美国CPA）的活跃会员进行一次性的职业诚信宣誓
- 该誓言旨在提高审计职业质量、降低舞弊风险
- 副作用：部分公司的CEO和CFO（持有会计学位者）也需要宣誓

**誓言内容**：
宣誓者承诺遵守职业行为准则，维护职业诚信，将公众利益置于个人利益之上。

**不遵守后果**：

- 失去执业资格
- 潜在声誉成本

### 1.2 双重差分（DiD）设计

**研究设计**：

- **处理组**：CEO或CFO持有荷兰专业会计学位、需要宣誓的公司（约15%的样本）
  - 40位CEO
  - 84位CFO
  - 共120家公司
- **对照组**：CEO和CFO不需要宣誓的公司
  - 717家公司
- **总样本**：837家公司（47家上市公司，790家私有公司）
- **时间窗口**：宣誓前后各3年（3年pre & 3年post）

**识别策略**：

- 准自然实验：在获得会计学位时誓言尚未存在，消除了自选择效应
- 誓言是意外的，当高管获得会计学位时并不知道自己将来需要宣誓
- 处理组和对照组之间的差异在于高管是否持有荷兰会计学位

## 二、应计盈余管理测度方法

### 2.1 Modified Jones模型（Dechow et al., 1995）

**模型选择理由**：
Heese et al. (2023) 采用Modified Jones模型作为主要的应计盈余管理测度方法，因为：

1. Dechow et al. (1995) 比较了五种常用的应计模型，发现Modified Jones模型效果最好
2. 该模型在会计研究中被广泛采用，具有良好的可比性
3. 能够有效分离正常应计和异常应计

**第一步：估计行业-年度横截面回归**

$$\frac{TA_{i,t}}{AT_{i,t-1}} = \alpha_0 + \alpha_1\frac{1}{AT_{i,t-1}} + \alpha_2\frac{\Delta REV_{i,t} - \Delta REC_{i,t}}{AT_{i,t-1}} + \alpha_3\frac{PPE_{i,t}}{AT_{i,t-1}} + \varepsilon_{i,t}$$

**变量定义**：


| 变量                  | 定义                                         |
| ------------------- | ------------------------------------------ |
| $TA_{i,t}$          | 公司i在第t年的总应计 = 净利润 - 经营活动现金流                |
| $AT_{i,t-1}$        | 公司i在第t-1年的总资产（滞后一期总资产）                     |
| $\Delta REV_{i,t}$  | 公司i在第t年的营业收入变化 = $REV_{i,t} - REV_{i,t-1}$ |
| $\Delta REC_{i,t}$  | 公司i在第t年的应收账款净额变化                           |
| $PPE_{i,t}$         | 公司i在第t年的固定资产原值（不动产、厂房和设备）                  |
| $\varepsilon_{i,t}$ | 回归残差                                       |


**第二步：计算非操控性应计（NDA）**

使用第一步估计的系数，计算每个公司-年度的正常（非操控性）应计：

$$NDA_{i,t} = \hat{\alpha}*0 + \hat{\alpha}1\frac{1}{AT{i,t-1}} + \hat{\alpha}2\frac{\Delta REV{i,t} - \Delta REC*{i,t}}{AT_{i,t-1}} + \hat{\alpha}*3\frac{PPE*{i,t}}{AT_{i,t-1}}$$

**第三步：计算操控性应计（DA）**

$$DA_{i,t} = \frac{TA_{i,t}}{AT_{i,t-1}} - NDA_{i,t}$$

**关键改进（与原始Jones模型的区别）**：

- Modified Jones模型在收入变化项中减去了应收账款变化（$\Delta REC_{i,t}$）
- 这一调整基于假设：应收账款的变化可能源于收入操纵
- 通过扣除应收账款变化，模型更好地捕捉了信用销售中的盈余管理

### 2.2 收入增加的操控性应计

**核心因变量**：
Heese et al. (2023) 关注**收入增加的操控性应计**（Income-Increasing Discretionary Accruals），即正的DA值：

$$DA^{+}*{i,t} = \max(DA*{i,t}, 0)$$

**研究发现**：

- CFO宣誓后，收入增加的操控性应计平均减少0.031
- 这相当于标准差的约0.26倍
- 约80%的CFO宣誓公司将应计盈余管理降低了20%以上

## 三、真实活动盈余管理测度方法

### 3.1 Roychowdhury (2006) 模型

**模型选择理由**：

1. 真实活动盈余管理比应计盈余管理更难被审计师和监管者发现
2. Roychowdhury模型能够捕捉三种主要的真实活动操纵方式
3. 可以与应计盈余管理进行比较，检验是否存在替代效应

**模型框架**：
真实活动盈余管理是指管理者偏离正常经营实践，通过实际经营活动来操纵盈余的行为。

### 3.2 异常生产成本（Abnormal Production Costs）

**估计模型**：

$$\frac{PROD_{i,t}}{AT_{i,t-1}} = \alpha_0 + \alpha_1\frac{1}{AT_{i,t-1}} + \beta_1\frac{REV_{i,t}}{AT_{i,t-1}} + \beta_2\frac{\Delta REV_{i,t}}{AT_{i,t-1}} + \beta_3\frac{\Delta REV_{i,t-1}}{AT_{i,t-1}} + \varepsilon_{i,t}$$

**变量定义**：


| 变量                   | 定义                      |
| -------------------- | ----------------------- |
| $PROD_{i,t}$         | 生产成本 = 销售成本（COGS）+ 存货变动 |
| $REV_{i,t}$          | 营业收入                    |
| $\Delta REV_{i,t}$   | 当年营业收入变化                |
| $\Delta REV_{i,t-1}$ | 上一年营业收入变化               |
| $AT_{i,t-1}$         | 滞后一期总资产                 |


**经济含义**：

- 管理者可能通过过度生产来降低单位成本，从而虚增利润
- 异常生产成本为正表示可能存在向上操纵

**研究发现**：

- CFO宣誓后，异常生产成本显著降低（系数为-0.063，p<0.05）
- 相当于标准差的约0.12倍

### 3.3 异常可自由支配费用（Abnormal Discretionary Expenses）

**估计模型**：

$$\frac{DisExp_{i,t}}{AT_{i,t-1}} = \alpha_0 + \alpha_1\frac{1}{AT_{i,t-1}} + \beta_1\frac{REV_{i,t-1}}{AT_{i,t-1}} + \varepsilon_{i,t}$$

**变量定义**：


| 变量             | 定义                                        |
| -------------- | ----------------------------------------- |
| $DisExp_{i,t}$ | 可自由支配费用 = 研发费用（R&D）+ 广告费用 + 销售及管理费用（SG&A） |
| $REV_{i,t-1}$  | 上一年营业收入                                   |
| $AT_{i,t-1}$   | 滞后一期总资产                                   |


**经济含义**：

- 管理者可能削减研发、广告等可自由支配费用来短期提高利润
- 异常可自由支配费用为负表示可能存在向上操纵

**研究发现**：

- CFO宣誓后，异常可自由支配费用显著降低（系数为-0.070，p<0.05）
- 相当于标准差的约0.13倍

### 3.4 综合REM指标

**综合指标构建**：

$$REM_{i,t} = -ABCFO_{i,t} + ABPROD_{i,t} - ABDisExp_{i,t}$$

或等价地：

$$REM_{i,t} = ABPROD_{i,t} - ABDisExp_{i,t}$$
（当使用异常CFO时，符号相应调整）

**研究发现**：

- CFO宣誓后，综合REM指标显著降低（系数为-0.139，p<0.05）
- 约80%的CFO宣誓公司将真实活动盈余管理降低了15%以上

### 3.5 关键发现：无替代效应

**重要结论**：
Heese et al. (2023) 发现：

1. 宣誓公司同时减少了应计盈余管理和真实活动盈余管理
2. **没有发现**应计盈余管理与真实活动盈余管理之间的替代效应
3. 这表明宣誓高管遵循的是法律的**精神**（spirit of the law），而不仅仅是**字面**（letter of the law）

## 四、主回归方程

### 4.1 基准回归模型

**主回归方程**：

$$Earnings\ Management_{i,t} = \beta_0 + \beta_1 Treatment_{i,t} + \phi Controls_{i,t} + \gamma_i + \delta_t + \varepsilon_{i,t}$$

**变量定义**：


| 变量                           | 定义                                  |
| ---------------------------- | ----------------------------------- |
| $Earnings\ Management_{i,t}$ | 盈余管理测度（DA+、REM_Prod、REM_DisExp、REM） |
| $Treatment_{i,t}$            | 处理变量：当公司i的高管在t年需要宣誓时=1，否则=0         |
| $Controls_{i,t}$             | 控制变量向量                              |
| $\gamma_i$                   | 公司固定效应（Firm FE）                     |
| $\delta_t$                   | 年度固定效应（Year FE）                     |
| $\varepsilon_{i,t}$          | 误差项                                 |


### 4.2 处理变量定义

**Treatment变量**：

- **CEO处理组**：CEO持有荷兰专业会计学位、需要宣誓的公司
- **CFO处理组**：CFO持有荷兰专业会计学位、需要宣誓的公司

**关键发现**：

- 当CFO宣誓时，盈余管理显著降低（系数显著为负）
- 当仅CEO宣誓时，没有显著效果
- 这与CFO负责日常财务活动、CEO更关注战略方向的现实一致

### 4.3 控制变量

**标准控制变量**：


| 变量           | 定义             | 预期符号 |
| ------------ | -------------- | ---- |
| Size         | 公司规模（总资产的自然对数） | 不确定  |
| Leverage     | 杠杆率（总负债/总资产）   | +    |
| ROA          | 资产回报率（净利润/总资产） | 不确定  |
| MTB          | 市账比（市值/账面价值）   | 不确定  |
| Sales Growth | 销售增长率          | 不确定  |
| Firm Age     | 公司年龄           | -    |
| Loss         | 亏损虚拟变量（当年亏损=1） | +    |


### 4.4 固定效应设置

**模型设定**：

- **公司固定效应（Firm FE）**：控制不随时间变化的公司异质性
- **年度固定效应（Year FE）**：控制宏观经济冲击对所有公司的共同影响
- **标准误聚类**：在公司层面聚类标准误

**识别来源**：

- 处理效应的识别来自同一公司在宣誓前后的变化
- 以及处理组与对照组之间变化的差异

## 五、稳健性检验方法

### 5.1 替代应计模型

#### 5.1.1 Dechow-Dichev (2002) 模型

**模型方程**：

$$\Delta WC_{i,t} = \beta_0 + \beta_1 CFO_{i,t-1} + \beta_2 CFO_{i,t} + \beta_3 CFO_{i,t+1} + \varepsilon_{i,t}$$

**变量定义**：


| 变量                | 定义                                                  |
| ----------------- | --------------------------------------------------- |
| $\Delta WC_{i,t}$ | 营运资本变动 = 应收账款变动 + 存货变动 - 应付账款变动 - 应交税费变动 + 其他流动资产变动 |
| $CFO_{i,t-1}$     | 上一年经营活动现金流                                          |
| $CFO_{i,t}$       | 当年经营活动现金流                                           |
| $CFO_{i,t+1}$     | 下一年经营活动现金流                                          |


**应计质量度量**：

- 使用回归残差的标准差作为应计质量的度量
- 标准差越大，应计质量越低
- Heese et al. (2023) 使用此模型作为稳健性检验，结果保持一致

#### 5.1.2 总应计模型（Chen et al., 2018）

**方法**：
Chen et al. (2018) 建议的两阶段估计方法：

1. 第一阶段：用Modified Jones模型估计残差
2. 第二阶段：将第一阶段残差对所有第二阶段回归变量和第一阶段回归变量进行回归

**目的**：

- 解决传统两阶段估计中的系数偏差问题
- Heese et al. (2023) 采用此方法，主结果依然稳健

#### 5.1.3 Kothari et al. (2005) 业绩匹配方法

**模型方程**：

$$\frac{TA_{i,t}}{AT_{i,t-1}} = \alpha_0 + \alpha_1\frac{1}{AT_{i,t-1}} + \alpha_2\frac{\Delta REV_{i,t} - \Delta REC_{i,t}}{AT_{i,t-1}} + \alpha_3\frac{PPE_{i,t}}{AT_{i,t-1}} + \alpha_4 ROA_{i,t-1} + \varepsilon_{i,t}$$

**关键改进**：

- 加入了滞后一期ROA作为控制变量
- 控制业绩对正常应计的影响
- 减少业绩差异导致的模型误设

### 5.2 其他稳健性检验

#### 5.2.1 增强识别策略

**安慰剂检验1：随机分配CFO**

- 随机将会计师CFO分配给公司
- 没有发现显著效果
- 排除了随机因素导致的虚假效应

**安慰剂检验2：比利时对照**

- 使用BoardEx识别比利时公司中具有会计背景的高管
- 比利时没有类似的宣誓要求
- 没有发现显著效果
- 排除了会计背景本身的影响

#### 5.2.2 替代对照组

**方法1：倾向得分匹配（PSM）**

- 使用PSM确定对照组观测值
- 在行业和规模等特征上进行匹配
- 结果依然稳健

**方法2：非活跃会计师对照**

- 对照组包括持有非活跃会计学位或商业背景的高管所在公司
- 结果依然稳健

#### 5.2.3 排除混淆因素

**检验1：同期事件**

- 排除荷兰公司治理准则变化的影响
- 结果依然稳健

**检验2：仅使用私有公司**

- 排除上市公司特有因素的影响
- 结果依然稳健

**检验3：排除无形资产和异常收益**

- 排除拥有大量无形资产的公司
- 排除有异常收益的公司
- 结果依然稳健

**检验4：替代时间窗口**

- 剔除2016年（政策实施当年）
- 结果依然稳健

#### 5.2.4 M-Score检验

**Beneish (1999) M-Score**：

- 综合的财务重述概率度量
- 检验极端报告选择
- CFO宣誓公司的M-Score降低0.139（约5.5%）

## 六、Heese方法的创新之处

### 6.1 研究设计创新

#### 6.1.1 准自然实验设计

**优势**：

1. **消除自选择偏差**：誓言在高管获得会计学位时并不存在，高管无法基于誓言预期选择是否获得会计学位
2. **外生冲击**：誓言要求是外生的政策变化，不受公司特征影响
3. **双重差分**：通过比较处理组和对照组在政策前后的变化，有效控制时间趋势和组间差异

#### 6.1.2 双重差分设计的有效性

**平行趋势假设**：

- 如果没有誓言政策，处理组和对照组的盈余管理趋势应该相似
- 论文通过事件研究法检验平行趋势，结果支持该假设

**识别假设**：

- 持有会计学位的高管所在公司与不持有会计学位的高管所在公司，在没有政策的情况下，盈余管理行为相似
- 论文通过多种稳健性检验支持这一假设

### 6.2 测度方法选择

#### 6.2.1 为什么选择Modified Jones模型

**与其他模型的比较**：


| 模型                    | 优点       | 缺点        | Heese的选择理由 |
| --------------------- | -------- | --------- | ---------- |
| Jones (1991)          | 简单、广泛使用  | 未考虑应收账款操纵 | 改进版更好      |
| Modified Jones (1995) | 考虑应收账款操纵 | 可能过度修正    | **最佳选择**   |
| Dechow-Dichev (2002)  | 基于现金流映射  | 需要多期数据    | 用于稳健性检验    |
| Kothari (2005)        | 业绩匹配     | 假设线性关系    | 用于稳健性检验    |


**Heese的选择**：

- Modified Jones模型是会计文献中最广泛接受的应计盈余管理测度
- 具有良好的检验力和设定合理性
- 结果易于与其他研究比较

#### 6.2.2 为什么同时检验应计和真实活动盈余管理

**理论意义**：

1. **完整性**：管理者可以通过多种方式操纵盈余，仅检验一种可能遗漏重要信息
2. **替代效应**：传统观点认为，当一种操纵方式受限时，管理者可能转向另一种方式
3. **行为洞察**：同时减少两种盈余管理表明高管遵循法律精神而非仅仅字面

**实证发现的意义**：

- 没有发现替代效应，说明誓言改变了高管的行为规范
- 这与行为经济学中关于道德提醒和承诺设备的理论一致

### 6.3 与之前文献的差异

#### 6.3.1 与以往盈余管理研究的差异


| 维度   | 传统研究          | Heese et al. (2023) |
| ---- | ------------- | ------------------- |
| 研究问题 | 盈余管理的存在性和决定因素 | 诚信誓言对盈余管理的影响        |
| 识别策略 | 横截面比较或时间序列    | 准自然实验（DiD）          |
| 因果推断 | 相关关系          | 因果关系                |
| 政策含义 | 描述性           | 明确的政策建议             |


#### 6.3.2 与以往誓言研究的差异


| 维度   | 以往研究（实验经济学） | Heese et al. (2023) |
| ---- | ----------- | ------------------- |
| 研究场景 | 实验室实验       | 真实市场数据              |
| 誓言类型 | 研究者设计的誓言    | 真实的职业誓言             |
| 样本规模 | 小样本         | 837家公司              |
| 外部效度 | 有限          | 高                   |


**与SEC宣誓条款研究的差异**：

- Griffin and Lont (2005), Bhattacharya et al. (2007) 研究美国SOX法案中的CEO/CFO宣誓要求
- Heese et al. (2023) 研究的是职业诚信誓言，而非特定财务报告的准确性宣誓
- 前者是法律合规要求，后者是职业道德承诺

### 6.4 研究设计的优势

#### 6.4.1 内部效度

**优势来源**：

1. **外生冲击**：誓言要求是外生的政策变化
2. **消除自选择**：高管无法基于誓言预期选择是否获得会计学位
3. **双重差分**：有效控制时间趋势和组间差异
4. **多种稳健性检验**：结果在各种检验下均保持稳健

#### 6.4.2 外部效度

**优势来源**：

1. **真实市场数据**：使用荷兰真实公司的财务数据
2. 包含上市公司和私有公司
3. 多种行业覆盖
4. 结果对政策制定具有直接参考价值

#### 6.4.3 政策相关性

**政策含义**：

1. 诚信誓言是一种**低成本**的治理机制
2. 相比增加执法力度或惩罚，誓言可能更具成本效益
3. 结果对监管机构、专业协会和公司治理具有启示意义

## 七、回归方程汇总

### 7.1 应计盈余管理估计方程（Modified Jones模型）

**第一阶段（行业-年度横截面回归）**：

$$\frac{TA_{i,t}}{AT_{i,t-1}} = \alpha_0 + \alpha_1\frac{1}{AT_{i,t-1}} + \alpha_2\frac{\Delta REV_{i,t} - \Delta REC_{i,t}}{AT_{i,t-1}} + \alpha_3\frac{PPE_{i,t}}{AT_{i,t-1}} + \varepsilon_{i,t}$$

**第二阶段（计算操控性应计）**：

$$DA_{i,t} = \frac{TA_{i,t}}{AT_{i,t-1}} - \left(\hat{\alpha}*0 + \hat{\alpha}1\frac{1}{AT{i,t-1}} + \hat{\alpha}2\frac{\Delta REV{i,t} - \Delta REC*{i,t}}{AT_{i,t-1}} + \hat{\alpha}*3\frac{PPE*{i,t}}{AT_{i,t-1}}\right)$$

**收入增加的操控性应计**：

$$DA^{+}*{i,t} = \max(DA*{i,t}, 0)$$

### 7.2 真实活动盈余管理估计方程（Roychowdhury模型）

**异常生产成本**：

$$\frac{PROD_{i,t}}{AT_{i,t-1}} = \alpha_0 + \alpha_1\frac{1}{AT_{i,t-1}} + \beta_1\frac{REV_{i,t}}{AT_{i,t-1}} + \beta_2\frac{\Delta REV_{i,t}}{AT_{i,t-1}} + \beta_3\frac{\Delta REV_{i,t-1}}{AT_{i,t-1}} + \varepsilon_{i,t}$$

$$REMProd_{i,t} = \hat{\varepsilon}_{i,t}$$

**异常可自由支配费用**：

$$\frac{DisExp_{i,t}}{AT_{i,t-1}} = \alpha_0 + \alpha_1\frac{1}{AT_{i,t-1}} + \beta_1\frac{REV_{i,t-1}}{AT_{i,t-1}} + \varepsilon_{i,t}$$

$$REMDisExp_{i,t} = \hat{\varepsilon}_{i,t}$$

**综合REM指标**：

$$REM_{i,t} = REMProd_{i,t} - REMDisExp_{i,t}$$

### 7.3 主回归方程

**基准模型**：

$$Earnings\ Management_{i,t} = \beta_0 + \beta_1 Treatment_{i,t} + \phi Controls_{i,t} + \gamma_i + \delta_t + \varepsilon_{i,t}$$

**展开形式（以DA+为例）**：

$$DA^{+}*{i,t} = \beta_0 + \beta_1 CFOOath*{i,t} + \beta_2 Size_{i,t} + \beta_3 Leverage_{i,t} + \beta_4 ROA_{i,t} + \beta_5 MTB_{i,t} + \beta_6 Sales\ Growth_{i,t} + \gamma_i + \delta_t + \varepsilon_{i,t}$$

### 7.4 稳健性检验方程

**Dechow-Dichev (2002) 模型**：

$$\Delta WC_{i,t} = \beta_0 + \beta_1 CFO_{i,t-1} + \beta_2 CFO_{i,t} + \beta_3 CFO_{i,t+1} + \varepsilon_{i,t}$$

**Kothari (2005) 业绩匹配模型**：

$$\frac{TA_{i,t}}{AT_{i,t-1}} = \alpha_0 + \alpha_1\frac{1}{AT_{i,t-1}} + \alpha_2\frac{\Delta REV_{i,t} - \Delta REC_{i,t}}{AT_{i,t-1}} + \alpha_3\frac{PPE_{i,t}}{AT_{i,t-1}} + \alpha_4 ROA_{i,t-1} + \varepsilon_{i,t}$$

## 九、研究结论与启示

### 9.1 主要结论

1. **应计盈余管理**：CFO宣誓后，收入增加的操控性应计显著降低（约0.26个标准差）
2. **真实活动盈余管理**：CFO宣誓后，异常生产成本和异常可自由支配费用均显著降低
3. **无替代效应**：没有发现应计盈余管理与真实活动盈余管理之间的替代效应
4. **CFO vs CEO**：效应主要集中在CFO宣誓的公司，CEO宣誓没有显著效果

### 9.2 理论贡献

1. **行为经济学理论**：为道德提醒和承诺设备的有效性提供了大样本实证证据
2. **盈余管理文献**：首次系统检验诚信誓言对财务报告质量的影响
3. **公司治理文献**：提供了一种低成本治理机制的有效性证据

### 9.3 实践启示

1. **政策制定**：诚信誓言可能是一种成本效益较高的监管工具
2. **专业协会**：可以考虑在职业准入中引入诚信宣誓要求
3. **公司治理**：公司可以考虑在内部推行高管诚信承诺机制
